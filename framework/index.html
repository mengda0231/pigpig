Motherboard

先有骨架，后有观点。

生猪行业不是靠多读几份周报就能看懂的行业。真正有用的研究框架，必须把主链、 相邻链、盈利模式、变量时钟和失真条件放到同一张图上，然后再用阶段切换去决定 当前哪些逻辑最有解释力。

Use rule 材料是输入，不是答案。

公开版默认把任何材料先当成假设输入，再问它落在主链哪一环、在什么条件下有效、 以及什么情况会失真。

Guardrail 公司材料不能单独改写行业状态。

公司是行业变量的映射对象，不是行业状态的定义者。只有当公司现象被主链变量和更广 样本验证，才有资格上升为行业含义。

Main chain

行业主链

生猪行业底层仍是一条供给驱动链，但研究的难点在于：不同阶段，主链里最重要的传导 环节会发生切换，不能永远只盯一个变量。

Slow layer

能繁母猪与补栏

能繁母猪决定未来供给方向，仔猪价格和补栏意愿决定中期供给安排。这是行业最慢、 也最容易被误读成“唯一答案”的一层。

Flow layer

出栏、均重与行为变量

出栏量给出现实供给，出栏均重、标肥价差和二育决定短周期压力释放的速度。很多 “现货为什么不涨”的答案都藏在这里。

Result layer

猪价、盈利与下一轮行为

猪价和单头盈利是结果变量，同时又通过亏损深度反过来影响母猪淘汰、补栏和扩张， 形成下一轮供给路径。

Adjacent chains

相邻链如何接入主链

饲料、疫病、政策和下游吸收都重要，但只有通过明确的 transmission path 接入主链， 它们才是研究变量，而不是平行叙事。

Feed cost

玉米 / 豆粕 -> 完全成本

成本端决定亏损深度、去化速度和公司分化，不单独决定猪价方向。它更像约束器和 加速器，而不是长期方向盘。

Disease shock

疫病 / 生物安全 -> 供给节奏

在平稳年份，它更像风险放大器；在 ASF 这类冲击阶段，它会直接改变主链的供给、 流通和恢复顺序。

Policy

政策与行为约束

收储、预警、降重导向和调运规则通常更影响节奏和振幅，而不直接取代长期供需方向。

Demand

屠宰 / 冻品 / 进口 -> 吸收能力

下游吸收更多解释“为什么现在不涨”或“为什么没继续跌”。它能托底短期价格，但 很少单独创造长期景气。

Profit engine

盈利模式不是只有猪价

行业最粗的表达式是 单头盈利 = 猪价 - 完全成本，但实际研究需要把价格、 成本、出栏兑现和负债表约束一起看。

Price layer

价格弹性

猪价和区域价差决定盈利方向，是主链里最显眼的结果变量，也是市场最容易过度关注的一层。

Cost layer

成本曲线

原料、生产成绩、折旧、财务费用和生物安全一起决定谁的亏损更浅、谁能在底部阶段留下来。

Volume layer

兑现与节奏

龙头出栏兑现、均重、二育和库存吸收决定短周期张力，很多“预期差”实际上就来自这层。

Indicators

指标层级

研究不能只看某一个指标，而要先判断它属于哪个时钟，再决定它能改写哪一层结论。

Leading / 领先变量

  • 能繁母猪存栏
  • 母猪淘汰量与淘汰折价
  • 仔猪价格与补栏意愿
  • 玉米、豆粕
  • 龙头月度出栏计划与兑现度

Coincident / 同步变量

  • 全国猪价与区域价差
  • 出栏量
  • 出栏均重
  • 标肥价差与二育活跃度
  • 屠宰量、开工率与冻品库存

Lagging / 滞后变量

  • 行业利润率
  • 公司现金流与财报修复
  • 一致预期上修
  • 分红能力与估值切换

Post-hoc / 后验与噪音

  • 只根据盘面解释基本面
  • 把周报观点当成事实
  • 用单一公司月报外推行业状态
  • 把事件标题直接等于主链变化
Distortion checklist

最常见的失真方式

真正危险的不是“数字记错”，而是把本来只在某个阶段成立的逻辑，写成跨阶段、跨层级 的普遍规律。

把价格、成本和股票混成一句话

价格决定利润方向，成本决定亏损深度，股票交易远期概率。三者不能被压扁成一句 “猪价跌所以股价也一定弱”。

把短期行为变量写成长期规律

均重、二育、冻品会改写节奏，但通常不单独决定长期方向。把它们写成“新周期定律” 往往会过度外推。

把公司例子越权写成行业状态

公司是行业变量的映射器，不是行业状态本身。单一龙头的经营变化，只有被主链变量和 更广样本验证后，才能上升为行业结论。

Conclusion ladder

结论层级

公开版沿用三层判断结构。它不是文风偏好，而是为了让跟踪系统知道：什么现在可以放心用， 什么只能带条件使用，什么还只能作为 hypothesis to test。

Tier A

硬底板

供给驱动方向、多时钟并行、股票领先利润兑现、成本改变亏损深度、公司不能改写行业、 交易层只能做镜像。

Tier B

条件性结论

能繁母猪是最重要慢变量锚、规模化强化龙头优势、低成本带来估值韧性、深亏损推动去化、 政策改变振幅、LH 有助于提前判断拐点。

Tier C

待验证命题

周期稳定缩短、波动永久下降、质量完全取代猪价、当前已确认进入加速去化、新范式已经成立。

框架页对应的是整个专题的 motherboard。后续所有公开更新都默认先回到这套结构里，再决定 哪些内容值得被写成更强结论。
