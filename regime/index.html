Regime map

同一个行业，不止一种读法。

生猪研究最常见的错误，不是记错数字，而是忽略阶段切换。正常年份里好用的逻辑， 到 ASF 冲击期可能失真；后 ASF 重建阶段成立的成长叙事，也不能自动推广为近两年的 “新范式”定律。

Bottom line 没有永远正确的单条逻辑，只有阶段内更有解释力的逻辑组合。

阶段划分不是把行业切死，而是帮助我们先问清：当前到底是哪种环境，哪些链条仍成立， 哪些链条必须重写。

Four phases

四阶段重建

这套阶段切换不是历史复述，而是研究顺序。它告诉我们：哪些变量在什么时候最重要， 股票在交易什么，以及哪类错误最容易在该阶段发生。

Phase 1 · 2016-07 to 2018-07

正常年份 baseline

供给错配导致利润率周期，能繁母猪、补栏、出栏量和均重决定价格方向。股票通常领先 现货与盈利兑现。

  • 最重要矛盾：慢变量与价格弹性
  • 最好用变量：能繁、仔猪、出栏、猪粮比
  • 最容易错：把正常年份的映射硬套到冲击期
Phase 2 · 2018-10 to 2020-04

ASF 结构断点

疫病、调运和政策把原本平滑的供给链打断。全国价格失真，区域价差和生物安全能力 权重大幅上升。

  • 最重要矛盾：真实供给变化与价格失真并存
  • 新增关键变量：疫病、调运、区域价差、复养能力
  • 最容易错：把高价格简单写成普通景气上行
Phase 3 · 2021-10 to 2023-08

后 ASF 重建

行业仍有周期属性，但开始明显混入成长、成本曲线和资产负债表分化。龙头月度出栏增速 变得越来越像行业变量。

  • 最重要矛盾：周期仍在，但成长与负债表主导分化
  • 最好用变量：成本曲线、月度出栏、现金流与负债
  • 最容易错：把所有猪企继续当成同一种 price beta 资产
Phase 4 · 2025-01 to 2026-04

近现实范式测试

周期没有死，但振幅、节奏和权益映射被效率、集中度、政策约束和 LH 镜像共同重塑。

  • 最重要矛盾：旧周期仍在，但表现形式被重塑
  • 新增重点：效率、龙头份额、政策约束、交易镜像
  • 最容易错：把“周期变形”写成“周期消失”
Compare

哪些规律延续了，哪些被改写了

阶段切换真正要回答的，不是“新旧谁对谁错”，而是旧逻辑哪些仍然有用，哪些需要更严格 的条件，哪些已经必须换一种问题意识来研究。

延续的底层规律

  • 供给仍决定行业方向
  • 股票通常领先现货与盈利兑现
  • 成本仍决定亏损深度与穿越能力
  • 去化依旧是中周期反转的必要路径

被改写的研究重心

  • 不能只看能繁母猪，要把效率与兑现度并入供给框架
  • 不能只看猪价，要把现金流、负债表和成长兑现一起看
  • 不能忽略政策对短周期振幅和行为变量的影响
  • 不能把 LH 和股票的领先理解成现实确认

仍在验证的变化

  • 周期长度是否已经稳定缩短
  • 价格波动是否会长期、永久性下降
  • 质量溢价是否已经取代价格弹性
  • 新范式是否真的足够稳，能成为硬底板
Switch rules

什么时候必须切换逻辑

阶段切换不是抽象概念，而是具体的研究动作。出现下面这些信号时，旧框架就不能继续 不加修正地使用。

从 price-led 到 cost-led

当猪价已经不足以解释公司股价差异，而低成本、现金流和分红能力开始主导定价时， 研究重点就必须切到成本曲线和质量分层。

从 sow-only 到 sow-plus-efficiency

当母猪变化无法单独解释真实供给，就必须把效率、专业分工、MSY 和龙头兑现一起纳入 供给框架。

从 event headline 到 transmission path

遇到疫病、政策、进口和调运事件时，不能直接写观点，必须先问它到底改变了主链里的 哪一个 transmission step。

阶段页的意义，是防止研究回到“谁最新谁正确”的状态。任何新材料都应先定位阶段，再决定 它是在修正当前阶段，还是只是在提供一个待验证的新假设。
