Current state desk

现在更像“去化必要性上升”，还不是“拐点现实已确认”。

截至 2026-04-12，我们把行业放在 phase 4 current paradigm test 里看：慢变量已经下行， 深亏损已经足够痛，但效率提升、集中度抬升和龙头月报兑现，仍在延后 供给收缩向现实价格的传导。所以当前更接近“底部前的验证区”，而不是 可以直接把反转写成硬结论的确认区。

State summary
当前判断 去化必要性上升，但现实性仍待确认。

官方能繁、利润压力和龙头月报放在一起看，当前最稳的说法仍然是： 行业已经进入更痛的验证区，但还不能把“现实供给已经明显收缩” 作为既成事实。

现在不能直接说 “猪价已见底”或“新范式已经定型”。

这两句话目前都会越权。因为慢变量虽然在下，效率还在抵消，短周期 的均重、二育、屠宰吸收和冻品链也还没有全部完成硬绑定。

下一步最该盯 母猪、效率、月报兑现和短周期吸收能不能共振。

只有当淘母、母猪、效率扩张、龙头月报、均重、二育和现货一起往 同一个方向走，判断才有资格往上升级。

State board

四块状态板

这四块板分别回答：供给基础有没有明显缩、利润压力够不够深、效率还在 不在抵消、以及市场镜像能不能当现实来用。它们对应的是当前行业最容易 被混淆的四个层面。

供给基础

高位回落，但不是经典底部式塌缩

2025-12-31 官方锚点仍显示： 能繁 3961 万头、 存栏 42967 万头、 出栏 71973 万头。

这意味着慢变量已经下台阶，但供给现实仍不属于“快速出清完成”。

利润压力

深亏损已经是必要条件，但还不是自动触发器

最新官方利润月报到 2025-08： 猪粮比 5.94，散养 -109 元/头，规模场 -57 元/头。

这足以强化“去化必要性”，但还不足以单独推出“去化已经加速兑现”。

效率 overlay

效率仍在抵消一部分母猪回落

当前阶段不能只看母猪。已学到的公司样本里， PSY / 成活率 / 上市率 仍在抬升， 解释了为什么同样的母猪基数下，头部出栏并没有按旧周期那样快收缩。

这也是 phase 4 相对旧周期最大的阅读升级点。

市场镜像

股票和 LH 只能先当镜像层

股票可以先行，LH 也可以先反映远期修复预期，但它们只能 写成 inferred，不能伪装成现实供给已经确认的 extracted 结论。

当前所有“盘面先动”都必须回到母猪、月报和短周期变量上复核。

Cycle ruler

行业阶段标尺

这根标尺不是时间线装饰，而是研究问法的切换器。当前我们明确站在 phase 4，看的是“旧周期被效率、政策、集中度和市场前置定价重塑后的 表现形式”，而不是在重复 baseline 的问法。

Phase 1 · 2016-07 至 2018-07

baseline

正常年份的猪周期语法。母猪、仔猪、补栏、出栏、利润和股价之间的 关系相对干净，效率变量的重要性还没有像今天这样抬升。

Phase 2 · 2018-10 至 2020-04

ASF break

疫病冲击是 regime break，不是普通事件。区域价差、生物安全、调运 约束和政策应对改变了旧规律的适用边界。

Phase 3 · 2021-10 至 2023-08

post-ASF rebuild

重建阶段里，规模化、集中度、成本曲线和负债表开始真正进入行业框架， “猪周期股”逐步混入“效率股”和“成长股”属性。

Phase 4 · 2025-01 至 2026-04

current paradigm test

供给不能只看母猪，必须一起看效率、龙头月报、均重、二育、库存吸收 和市场镜像。当前页默认就在这个问法里读所有变量。

Judgment ladder

结论分层

这层梯子用来提醒自己：哪些已经是研究底板，哪些必须带条件使用， 哪些现在最多只能当假设。可视化条不是概率预测，只是当前资格层级。

Tier map
Tier A · 供给仍是方向盘

这条是当前行业底板。需求、政策、盘面和成本都能影响节奏， 但长期方向仍然要回到供给。

Tier B · 深亏损强化去化，但要带条件用

亏损深、持续久、效率不再抵消、龙头月报同步转弱时，这条判断 才会更接近现实层。

Tier C · 新范式还没到硬规律

周期缩短、波动收敛、质量溢价抬升都存在机制基础，但还没有被 足够长样本和独立数据验证到可以当“常识”。

Read rule

现在最容易犯的错

第一类：把 母猪略降 直接读成未来供给必然明显收缩。

第二类：把 深亏损 直接读成去化现实已经启动。

第三类：把 股票 / LH 先动 直接读成基本面已确认。

这三类误判，本质上都来自把不同层级的证据混写。当前页会一直强制 拆开：official、company、 support、inferred。

Change the call

什么会改变现在的判断

这三张卡是升级判断的必要条件清单。只有它们开始同向共振，行业状态 才有资格从“验证区”升级到“更接近现实拐点”。

慢变量确认

母猪、淘母、更新量继续配合下行

如果后续官方母猪数据继续往下，淘母价格和淘汰量同步确认，而且补栏 与更新意愿没有明显反弹，慢变量层才会开始真正支持“供给收缩已在路上”。

效率不再抵消

头部效率扩张放缓，月报兑现转弱

只有当 PSY、成活率、上市率这类效率改善不再继续冲抵母猪回落，或者 龙头月报本身开始转弱，供给收缩才会从纸面慢变量传到现实层。

短周期共振

均重、二育、屠宰、冻品和现货一起改善

这一步决定“拐点是预期中的，还是现实中的”。如果均重回落、二育降温、 屠宰吸收改善、库存缓和和现货企稳能够同向出现，判断才会更扎实。

当前页默认采用的阅读顺序是： 母猪 -> 效率 -> 龙头月报 -> 短周期行为 -> 现货 -> 股票 / LH。 只要这条顺序被跳过，行业判断就容易重新滑回“看着很懂但不够可升级”。
