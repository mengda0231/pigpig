supply identity

母猪头数 x 效率 x 出栏兑现

当前阶段默认不用单一能繁母猪去外推未来供给，而是按下面这条式子去看：

真实商品猪供给 ~= 行业能繁母猪 x 公司母猪 / 后备母猪 x 生产效率 (PSY / MSY / 分娩率 / 成活率) x 出栏兑现

官方已绑定: 能繁母猪 公告已绑定: 龙头月报出栏 待结构化: 公司母猪 待结构化: 后备母猪 待结构化: PSY / MSY

read order

研究顺序

  • 先看行业能繁母猪是不是只是高位缓降，而不是断崖式去化。
  • 再看头部公司母猪和后备母猪是不是还在扩张或维持高位。
  • 再看 PSY、成活率和分娩率是不是继续提升。
  • 最后用月度出栏和高频均重、二育、屠宰吸收去验证近端压力。

binding rule

What stays honest

  • 母猪层默认只用官方月度序列。
  • 公司母猪、后备母猪、PSY 先作为公司披露层，不冒充统一官数。
  • 仔猪、均重、标肥、二育、屠宰、冻品在没统一前只作为高频验证代理。
  • 股票和 LH 继续只做 mirror，不越权改写主链。

current judgment

What the board supports right now

  • 能繁母猪边际回落存在，但仍更像高位缓降，不是断崖式去化。
  • 规模场单头利润已进入亏损区，但亏损本身不足以推出供给立刻收缩。
  • 屠宰同比仍强，说明供给现实和吸收节奏还没有给出清晰的紧平衡确认。
  • 温氏、新希望等已出现公司披露级效率指标，龙头效率差异可以开始进入供给判断。
  • 当前阶段最稳的写法仍是：母猪、效率、月报兑现和短周期高频必须并读。
